# Liocinema and Jotstar Strategic Merger Analysis

## Project Overview
This Power BI dashboard analyzes the strategic merger between Liocinema and Jotstar platforms.

The project provides insights into:
- User growth
- Subscriber behavior
- Content performance
- Watch time analysis
- Platform engagement
- Subscription trends

---

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling

---

## Dashboard Pages

### Home Page
Navigation page for all dashboards.

### Overview Dashboard
- Total Users
- Active vs Inactive Users
- City Tier Analysis
- Subscription Analysis

### Content Library Analysis
- Genre distribution
- Language analysis
- Content type comparison
- Runtime analysis

### Subscriber Analysis
- Subscription plan trends
- User age groups
- Upgrade/Downgrade analysis

### Content Consumption Analysis
- Watch time analysis
- Device usage
- Platform comparison
- Subscription watch trends

---

## Key Insights
- Tier 1 cities have the highest paid users.
- Premium plans generate maximum watch time.
- Mobile devices dominate content consumption.
- Movies are the most consumed content category.
